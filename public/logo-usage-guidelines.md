# The Internet Health Site - Logo Usage Guidelines

## Logo Overview
The Internet Health Site logo features an animated heart symbol that represents health, vitality, and the caring community aspect of our brand. The heart beats at a natural rhythm (approximately 60-80 beats per minute) to symbolize life and wellness.

## Logo Variations

### 1. Animated Heart Logo (Primary)
- **File**: `AnimatedHeartLogo.tsx`
- **Usage**: Primary logo for web applications and digital platforms
- **Animation**: Heartbeat animation with pulse ring effect
- **Sizes**: Small (32px), Medium (48px), Large (64px), Extra Large (96px)

### 2. Static Heart Logo (Secondary)
- **File**: `StaticHeartLogo.tsx`
- **Usage**: Print materials, favicons, and contexts where animation is not suitable
- **Layouts**: Horizontal, Vertical, Icon-only
- **Sizes**: Small (32px), Medium (48px), Large (64px), Extra Large (96px)

## Color Specifications

### Primary Colors
- **Heart Gradient Start**: #FF6B6B (Red-400)
- **Heart Gradient Middle**: #FF5252 (Red-500)
- **Heart Gradient End**: #E53E3E (Red-600)
- **Heart Border**: #B91C1C (Red-700)

### Text Colors
- **Primary Text**: #111827 (Gray-900)
- **Secondary Text**: #6B7280 (Gray-500)

## Usage Guidelines

### DO:
✅ Use the animated version for web applications and digital platforms
✅ Use the static version for print materials and small sizes
✅ Maintain proper spacing around the logo (minimum 16px clearance)
✅ Use on white or light backgrounds for optimal contrast
✅ Scale proportionally to maintain aspect ratio

### DON'T:
❌ Modify the heart shape or animation timing
❌ Change the color scheme without approval
❌ Use on backgrounds that reduce readability
❌ Stretch or distort the logo
❌ Add effects or filters to the logo

## Technical Specifications

### Web Implementation
- **Format**: React Component (TSX)
- **Animation Duration**: 1.2 seconds per heartbeat cycle
- **Animation Timing**: ease-in-out for natural movement
- **Browser Support**: All modern browsers
- **Performance**: Optimized CSS animations, no JavaScript required

### File Sizes
- **Animated Component**: ~2KB gzipped
- **Static Component**: ~1.5KB gzipped
- **CSS Animations**: ~0.5KB additional

## Accessibility
- The logo includes proper semantic markup
- Animation respects `prefers-reduced-motion` media query
- High contrast ratios maintained for readability
- Screen reader friendly with appropriate alt text
